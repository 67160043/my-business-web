## 1. 🏢 Logo.jpg - โลโก้บริษัท

### รายละเอียด
- **ชื่อไฟล์:** `Logo.jpg` (ต้องใช้ชื่อนี้เท่านั้น)
- **ขนาดที่แนะนำ:** 100x100 pixels (หรือ 200x200, 400x400 - สัดส่วน 1:1)
- **รูปแบบไฟล์:** JPG หรือ PNG (แนะนำ PNG ถ้าต้องการพื้นหลังโปร่งใส)
- **ขนาดไฟล์:** ไม่เกิน 500 KB

### ใช้ในหน้าไหนบ้าง
- ✅ index.html (หน้าแรก)
- ✅ about.html (เกี่ยวกับเรา)
- ✅ services.html (บริการ)
- ✅ contact.html (ติดต่อ)
- **ปรากฏที่:** มุมบนกลางของทุกหน้า (Header)

### คำแนะนำ
- ควรเป็นโลโก้ที่มีสัญลักษณ์เกี่ยวกับเทคโนโลยี
- สีที่เข้ากับธีม: น้ำเงิน, ฟ้า, เทอควอยส์, ขาว
- สไตล์: โมเดิร์น, มินิมอล, เรียบง่าย
- พื้นหลังโปร่งใส (PNG) จะดูดีที่สุด

### ตัวอย่างที่ควรมี
- รูปทรงเรขาคณิต (วงกลม, สี่เหลี่ยม, รูปดาว)
- สัญลักษณ์: เครือข่าย, Circuit board, คอมพิวเตอร์, ชิป
- ตัวอักษร "FT" หรือ "First Tech" ในรูปแบบโมเดิร์น

### หาโลโก้ได้ที่ไหน
- สร้างเองฟรี: https://www.canva.com/create/logos/
- สร้างฟรี: https://logomakr.com
- ดาวน์โหลด template: https://www.flaticon.com

## 📂 วิธีจัดเก็บรูปภาพ

### ตัวเลือก 1: วางรูปในโฟลเดอร์เดียวกับไฟล์ HTML (แนะนำ)
```
โฟลเดอร์โปรเจกต์/
├── index.html
├── about.html
├── services.html
├── contact.html
├── styles.css
├── Logo.jpg          ← วางที่นี่
├── oner.jpg          ← วางที่นี่
├── net.png           ← วางที่นี่
├── nett.png          ← วางที่นี่
└── nettt.png         ← วางที่นี่
```

### ตัวเลือก 2: สร้างโฟลเดอร์ images/
```
โฟลเดอร์โปรเจกต์/
├── index.html
├── about.html
├── services.html
├── contact.html
├── styles.css
└── images/
    ├── Logo.jpg
    ├── oner.jpg
    ├── net.png
    ├── nett.png
    └── nettt.png
```

**ถ้าเลือกตัวเลือก 2 ต้องแก้โค้ดในไฟล์ HTML ทุกหน้า:**

เปลี่ยนจาก:
```html
<img src="Logo.jpg" />
```

เป็น:
```html
<img src="images/Logo.jpg" />
```
