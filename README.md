First Technology Website

Project Overview

โปรเจกต์นี้เป็นตัวอย่างหน้าเว็บไซต์ของ First Technology
ทำขึ้นเพื่อฝึกสร้างเว็บเพจพื้นฐาน
ใช้เพื่อศึกษาการเขียนโครงสร้าง HTML ระดับเริ่มต้น
ไม่มีการใช้งาน CSS หรือ JavaScript ในเวอร์ชันนี้

โครงสร้างไฟล์
index.html – หน้าแรกของเว็บไซต์
about.html – หน้าข้อมูลบริษัท เรื่องราว และวัตถุประสงค์
services.html – หน้ารายการบริการ อธิบายสิ่งที่บริษัทให้บริการ
contact.html – หน้าสำหรับติดต่อ พร้อมข้อมูลตำแหน่งบริษัท
images/ – เก็บไฟล์รูปภาพที่ใช้ภายในเว็บไซต์
README.md – เอกสารอธิบายรายละเอียดโปรเจกต์นี้
